# hscMap3

[http://hscmap.mtk.nao.ac.jp/bleeding-edge/]() <= working demo


## Install
1. Install prerequisites
    * [Node](https://nodejs.org/)

1. Clone
    ```sh:install.sh
    git clone https://github.com/michitaro/hscMap.git
    ```

1. Build
    ```sh:build.sh
    cd frontend
    npm install
    npm run dev-server
    ```

1. Open local dev-server
    Open [http://localhost:8080]()
